
# SteamSort

**SteamSort** — это веб-сервис для тех, кто зарылся в своей библиотеке Steam. Приложение анализирует ваш профиль и сопоставляет игры с данными **HowLongToBeat**, помогая выбрать следующую игру для прохождения, исходя из свободного времени.

---

## Основные функции

* **Импорт библиотеки:** Мгновенная загрузка всех игр через Steam ID.
* **Сортировка по времени:** Узнайте, что можно пройти за вечер, а на что уйдет 100+ часов.
* **Фильтрация:** Поиск по названию и статусу прохождения.
* **Семейная библиотека:** добавье список игр из вашего Steam Family.
